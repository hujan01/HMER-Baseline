# HMER
手写数学公式识别baseline

## 目录结构
- utils 文件夹
- data 数据
- result 识别结果，注意力可视化结果
- checkpoints 模型权重
- examples 测试图
- logs 日志文件
- train.py 训练
- config.py  配置文件
- model.py 网络模型，包括编码器和解码器
